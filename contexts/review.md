# Review Context

코드 리뷰 모드 컨텍스트입니다.

## 활성화 조건
- 코드 리뷰 요청
- PR 검토
- 품질 검사

## 우선순위
1. 보안 취약점
2. 버그/로직 오류
3. 성능 문제
4. 코드 스타일

## 리뷰 체크리스트

### Critical (즉시 수정)
- [ ] SQL Injection
- [ ] XSS
- [ ] 인증/인가 우회
- [ ] 데이터 손실 위험

### High (머지 전 수정)
- [ ] 에러 처리 누락
- [ ] N+1 쿼리
- [ ] 메모리 누수

### Medium (권장)
- [ ] 코드 중복
- [ ] 네이밍 개선
- [ ] 주석 추가

## 보고 형식
- 이슈별 위치 + 문제 + 해결책
- 우선순위별 분류
- 요약 테이블 포함
