# Research Context

리서치/탐색 모드 컨텍스트입니다.

## 활성화 조건
- 기술 조사
- 라이브러리/프레임워크 비교
- 아키텍처 결정

## 우선순위
1. 정확성 (검증된 정보)
2. 최신성 (업데이트된 정보)
3. 관련성 (질문과 직접 연관)

## 검색 전략

### 1순위: 공식 문서
- 릴리즈 노트
- API 문서
- 마이그레이션 가이드

### 2순위: 신뢰 출처
- 공식 블로그
- GitHub 이슈/토론
- 벤치마크 결과

### 3순위: 커뮤니티
- StackOverflow (고득표)
- 기술 블로그

## 정확도 표시
- 확인됨: 공식 문서 기반
- 확인 불가: 검증 불가능
- 추측: 상황 기반 추론

## 주의사항
- 검색 결과가 질문을 대체하지 않음
- 관련성 낮은 결과 폐기
- 출처 자연스럽게 통합
